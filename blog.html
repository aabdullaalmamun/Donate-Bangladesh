Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content.

How do you select an element from the DOM?

Select an element from the DOM: 1. Using getElementById 2. Using getElementsByClassName 3.Using getElementsByTagName 4. Using querySelector 5.Using querySelectorAll

what is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where you add a single event listener to a parent element to handle events for its child elements. This works because of event bubbling, where events on child elements "bubble up" to their parents. It's useful for: Fewer Event Listeners: Instead of adding separate listeners to every child element, you use one listener on the parent, saving memory and improving performance. Handles New Elements: It works for dynamically added child elements without needing new listeners.

How do you manipulate an element's attributes and styles using the DOM?

1.Manipulating Attributes Attributes like id, class, src, href, alt, etc., can be manipulated using the following methods: getAttribute(), setAttribute(), removeAttribute();
2.Manipulating Styles CSS styles can be manipulated directly through the style property or by adding/removing CSS classes.
